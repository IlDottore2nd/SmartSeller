# Smart Seller

A simple terminal QoL mod for Lethal Company.

Type a target amount in the terminal and the mod automatically finds the closest combination of scrap to sell.

Works only at the Company Building after landing.

## Commands

```txt
sell 300
```

Example:
- `sell 300`
- `sell 700`
- `sell 120`

The mod:
- checks current company sell percentage
- calculates scrap values automatically
- finds the closest possible combination
- sells matching scrap from the ship

## Notes

- Host-side recommended
- Works best when host has the mod installed
- Clients are optional
- Built for BepInEx

## Installation

Place `SmartSeller.dll` inside:

```txt
BepInEx/plugins/
```
